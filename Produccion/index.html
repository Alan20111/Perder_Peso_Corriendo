Acerca del proyecto

Proyecto JavaFX (Gradle) en etapa de plantilla inicial: contiene la estructura base generada (aplicación, controlador y modelo), lista para implementar la lógica de un simulador de pérdida de peso por carrera.

Nota honesta: aún no incluye la lógica del simulador. La ventana mostrada es la plantilla real tal cual (etiqueta + botón “Hello!”).
